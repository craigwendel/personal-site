Projects

What's For Dinner?

This is a full stack application built with React.js, React Router, and Node.js. This app allows users to enter items they have at home and see recipes they can make with the ingredients they have. It ties into the Food2Fork API as well as a custom built API on the backend allowing users to register and log in.

Mock Apple Website

This project was built with React JS and React Router. It is a mock up of apple.com with products available to browse and add to a fictional cart. The syling was done from scratch to mimic the look and feel of apple.com

Gabble!

This project was built with node.js and Express. It allows users to sign up, create posts or "Gabs", and see what other people are posting. It is hosted on heroku and ties into a SQL database using Postgres.

SoundCloud

This project was built with vanilla JavaScript. This application uses the API from Soundcloud to fetch artists/tracks and dynamically inject HTML results into the DOM. This was our first major project using AJAX requests and I really enjoyed working on this one!
